# Casual Pokemon Team Builder

## Hosted at https://nqnt41.github.io/casual-pkmn-team-builder/

This project is intended as a way to help one design their ideal Pokemon video game team!

The initial page allows the user to select which Pokemon game to base their team on. From there, select a team of six Pokemon to form your team.

Up to five teams of Pokemon can be stored and accessed by team number.
